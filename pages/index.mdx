# Project:;COLD 攻略サイト（非公式）
 
このページはProject:;COLD（プロジェクト・コールド）シリーズのファン有志による非公式の攻略サイトです。Project:;COLDや第四境界の運営を行っている組織とは関係ございません。後から追いかける方々が攻略情報を集めやすくするため、[公式ガイドライン](https://www.project-cold.net/news/post-1/)に準拠して本サイトはネタバレを含みます。

また、このサイトは攻略サイトであって、このサイト内に謎が隠されていることはありません。ARGにおいてこのサイトを探索する必要はありません。

**メニュー・検索機能について**
- スマートフォン版で閲覧する場合、右上の **≡** ボタンを押すことでメニュー（ページ一覧）が表示されます。
- "Search documentation" ボックスに単語を入力することでサイト内検索が可能です。スマートフォン版では右上のメニューをタップすると検索窓が表示されます。
